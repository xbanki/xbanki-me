# Quick Start #

To build & run this repository, you need to have [Vue CLI](https://cli.vuejs.org/guide/) installed. This can be done by running either:

 - `yarn global add @vue/cli` or
 - `npm i -g @vue/cli`

Depending on your preferred package manager. After that, to run the repository locally do the following in order:

 - Clone this repository using your preferred command line tool and `cd path/to/xbanki-me`
 - run `yarn` & `yarn serve` to open up a development server
 - Open your favorite web browser & navigate to `localhost:8080` to see the application running.
